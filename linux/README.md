# Linux скрипты

Содержит утилиты и примеры для администрирования Linux: обновление системы, диагностика сети и рекомендации по безопасности.

Папки:

- `system/` — системные скрипты (например, `update-system.sh`)
- `network/` — сетевые утилиты (например, `net-info.sh`)
- `security/` — скрипты безопасности (например, `ssh-hardening.sh`)

Основные скрипты и примеры запуска:

- `linux/system/update-system.sh` — обновление системы (apt/dnf/yum):

```bash
sudo ./linux/system/update-system.sh
```

- `linux/network/net-info.sh` — краткая диагностика сети:

```bash
./linux/network/net-info.sh
```

- `linux/security/ssh-hardening.sh` — создание резервной копии `sshd_config` и рекомендации;
	для автоматического применения рекомендуемых настроек:

```bash
sudo ./linux/security/ssh-hardening.sh --apply
```

Рекомендации перед запуском скриптов:

- Всегда просматривайте содержимое скрипта перед выполнением.
- Для системных команд запускайте с `sudo` или от root.
- Тестируйте в безопасном окружении (контейнер/VM) перед применением в проде.

См. README в подпапках для деталей и дополнительных опций.
