Windows скрипты

Содержит скрипты для Windows (PowerShell, Batch и администрирование).

Основные скрипты в папках:

- `windows/powershell/` — PowerShell утилиты (например, get-disk-space.ps1)
- `windows/batch/` — Batch файлы (например, cleanup-temp.bat)
- `windows/administration/` — административные скрипты (например, create-local-user.ps1)

Пример запуска PowerShell (запуск от администратора рекомендуется):

```powershell
.\\windows\\powershell\\get-disk-space.ps1
```

Пример запуска Batch:

```bat
windows\\batch\\cleanup-temp.bat
```

Смотрите README внутри подпапок для деталей и дополнительных скриптов.
