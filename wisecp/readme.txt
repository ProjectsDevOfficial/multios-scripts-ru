
** WELCOME TO WISECP QUICK INSTALLER **

The quick installer will download the installation files for you from our license center and unzip on your server. Please follow the steps below.

1) Upload the installer.php file inside the zip file to the directory where you want to install WISECP.

2) Run it in the web browser address line as in the following example:

	www.yourdomain.com/installer.php

3) At this stage you can now continue with the quick installer.
